#  Memory Card Game

A fun and interactive Memory Card Game built using HTML, CSS, and JavaScript.

##  Objective
Match all pairs of cards by flipping two cards at a time.

##  How to Play
1. Click on a card to flip it.
2. Click another card to find its match.
3. If they match, they stay open.
4. If not, they flip back.
5. Match all pairs to win!

##  Features
- Move counter
- Timer
- New Game button
- Responsive design
- Attractive UI

##  How to Run
Just open `index.html` in your browser.

##  Technologies
HTML, CSS, JavaScript

Enjoy the game!
